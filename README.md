# 🌿 EcoAgro Verde — Sustentabilidade no Campo
**Concurso Agrinho 2026 · Subcategoria 3 — Ensino Médio**
Tema: *Agro forte, futuro sustentável* | Foco: Energia Limpa e Resíduos

## Sobre o Projeto
Site interativo sobre o uso de energia limpa (solar, eólica, biogás) e gestão de resíduos no campo, desenvolvido para o Concurso Agrinho 2026.

## Tecnologias
- HTML5 semântico
- CSS3 puro (variáveis, Flexbox, Grid, Media Queries)
- JavaScript puro (DOM, eventos, funções)

## Estrutura
```
ecoagro-verde/
├── index.html
├── README.md
├── css/style.css
└── js/script.js
```

## Como usar
1. Abra o `index.html` direto no navegador, OU
2. Suba no GitHub e ative o Pages em Settings → Pages

## Funcionalidades
- Quiz interativo com 8 perguntas e resultado personalizado
- Cards de dicas práticas por tema
- Galeria de fotos
- Menu responsivo com hambúrguer no celular
- Destaque automático do menu conforme a seção

## Personalização
Busque os comentários `⚠️ PERSONALIZE AQUI` no HTML e troque:
- Seu nome, escola, cidade e orientador(a)
- Sua mensagem pessoal sobre o tema

## Prompts de IA utilizados

**1. Pesquisa sobre energia solar no campo**
```
Quais são os principais benefícios da energia solar para pequenas
propriedades rurais no Brasil? Explica de forma simples.
```

**2. Ajuda com o quiz em JavaScript**
```
Como criar um quiz simples em JavaScript puro, sem bibliotecas,
que mostra perguntas uma por vez e dá um feedback no final?
```

**3. Dúvida sobre CSS Grid**
```
Como usar CSS Grid para criar uma grade de cards responsiva
que vira uma coluna no celular?
```

**4. Conteúdo sobre resíduos agrícolas**
```
Quais são as leis brasileiras sobre descarte de embalagens de
agrotóxico? Explica de forma simples para um trabalho escolar.
```

> A IA foi usada para tirar dúvidas pontuais. O conteúdo, as escolhas de design e a personalização são de autoria dos estudantes.
